# MarkIt v3 🎯

MarkIt is a lightweight image annotation tool for drawing, moving, and resizing bounding boxes and polygons.

## Features
- Draw rectangles and polygons
- Move and resize annotations
- Edit polygon vertices
- Double-click to change labels
- Export annotations as JSON

## Controls
- R → Rectangle tool
- V → Select tool
- P → Polygon tool
- Drag → Move / Resize
- Double Click → Change label

## How to Use
1. Open index.html in browser
2. Load images or folder
3. Annotate using tools
4. Export results
